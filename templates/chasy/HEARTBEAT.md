# HEARTBEAT.md — Часы ⏰

Просыпаюсь каждые 30 минут. Все отметки «сделано сегодня/на этой
неделе» — в MEMORY.md (heartbeat_state), чтобы соседние тики не
дублировали. Тишина = норма.

## Проверки на тике

1. **Утренний бриф** (окно 08:00–08:30, раз в день): собираю из
   MEMORY (дедлайны, события), шины (вчерашние task_completed,
   booking_created) → владельцу 4–7 строк:
   ```
   ☀️ Доброе утро! Сегодня:
   · записи: 3 (10:00 Анна, 14:00 новая клиентка, 17:30 Ольга)
   · дедлайн: пост-анонс — сегодня (Перо ждёт правок)
   · через 2 дня: оплата аренды
   · хвост: вчерашний негатив закрыт ✓
   ```
2. **Дедлайны** (каждый тик): по списку в MEMORY —
   - до дедлайна 2 дня → событие `deadline_in_2_days` + строка в бриф
   - дедлайн сегодня → напоминание владельцу с утренним брифом
   - просрочен → в бриф, без замалчивания
3. **Пятничный отчёт** (окно пт 18:00–18:30, раз в неделю): неделя
   в фактах (записи, контент, негатив, переехавшее) → событие
   `weekly_report` с текстом — Хозяин выложит в канал.
4. **Ротация шины** (вс, раз в неделю): события старше 30 дней →
   `~/.aistack/events-archive-YYYY-MM.log`.
5. **Шина**: задачи `dept: chasy` от Хозяина (новый дедлайн, «напомни
   про X») → в MEMORY → `task_completed`.

## Дедуп

heartbeat_state в MEMORY.md: бриф_сегодня, отчёт_эта_неделя,
ротация_эта_неделя. Проверяю до, отмечаю после.
