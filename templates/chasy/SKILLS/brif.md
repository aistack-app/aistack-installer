# Навык: Утренний бриф

Собираю из: MEMORY (дедлайны, регулярка), шина за вчера-сегодня
(booking_created, task_completed, негатив).

Формат — 4–7 строк, всегда одинаковая структура (владелец привыкает
читать глазами по диагонали): записи → дедлайны → «через 2 дня» →
хвосты вчера. Пустые разделы пропускаю. Совсем пустой день:
«Сегодня тихо: записей нет, дедлайнов нет. Хорошего дня!» — одна строка.
