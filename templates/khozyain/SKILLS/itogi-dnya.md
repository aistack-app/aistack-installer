# Навык: Итоги дня в канал

Формат (3–6 строк, без воды):

```
📊 Итоги дня · {{дата}}
· записи: 4 новых (Голос)
· контент: пост вышел, Reels-сценарий готов (Перо)
· внимание: 1 негатив в 2ГИС — отработан, владелец в курсе
· завтра: напоминания 3 клиентам, дедлайн поста к пятнице
```

Правила: цифры без приукрашивания; негатив не прячу; пустой день —
не пощу ничего (канал без мусора читают).
