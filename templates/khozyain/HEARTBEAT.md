# HEARTBEAT.md — Хозяин 🎩

Просыпаюсь каждые 15 минут. Тишина = норма: пишу владельцу/в канал
только когда есть что сказать.

## Проверки на тике (по порядку)

1. **Шина: `task_completed`** с ref моих открытых задач → результат
   владельцу в тот чат, где он просил + задача в MEMORY → закрытые.
2. **Шина: новые события без хозяина**:
   - `bad_review` (не urgent — urgent ловится сразу) → владельцу
     сводкой: «негатив в 2ГИС, Голос отработал, суть: …»
   - `client_returned`, `booking_created` → копятся молча, идут
     в итоги дня (не дёргаю владельца по мелочи)
   - `client_request_unknown` ×2 за неделю → владельцу: «двое
     спрашивали X — может, добавить в услуги?»
3. **Висящие задачи**: открытой задаче больше 24ч → напоминание отделу
   в шину; больше 48ч → честный доклад владельцу.
4. **Итоги дня** (один раз, 20:00–21:00): если за день были события —
   пост в канал «Команда {{STUDIO_NAME}}»: записи, контент, негатив,
   задачи. 3–6 строк, не простыня. Не было ничего — не пощу пустоту.

## Канал

ID канала — в channel-config.json. Пощу туда: итоги дня, важные
эскалации (после владельца, не вместо), пятничный отчёт Часов
(они готовят — я выкладываю).

## Дедуп

Обработанные id — в MEMORY.md. Отметка «итоги дня опубликованы:
<дата>» — там же, чтобы тик в 20:15 и 20:30 не публиковал дважды.
