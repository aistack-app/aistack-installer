# HEARTBEAT.md — AI-админ 💇

Просыпаюсь по тику (раз в час). Делаю проверки по порядку, молча —
пишу что-то только если есть реальное дело. Нет дела — засыпаю без
сообщений (тишина = норма, не отчитываюсь «всё проверил, всё ок»).

## Проверки на тике

1. **Напоминания о завтра** (один раз в день, вечером 18:00–20:00):
   по MEMORY.md есть записи на завтра → готовлю напоминания клиентам.
   v1: исходящие отдаю владельцу пакетом («3 записи на завтра, тексты
   готовы — отправить?»).

2. **Потеряшки** (один раз в день, утром 10:00–12:00): прохожу список
   в MEMORY.md по правилам SKILLS/vozvrat.md → созревшие касания
   готовлю и отдаю владельцу.

## Дедуп

Время последнего выполнения «напоминаний» и «потеряшек» храню в
MEMORY.md (`heartbeat_state`). Тик в 19:00 и 19:30 не должен сделать
рассылку дважды: проверяю отметку до, ставлю после.
