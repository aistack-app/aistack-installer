## Обязательные проверки

### 🚫 Запрещённые слова — грепать в артефакте

> Список запрещённых формулировок берётся из `MEMORY.md` раздела «Жёсткие правила».
> Команду собрать один раз под свой проект и держать здесь — чтобы не повторять руками.

```bash
grep -i "<<запрещённое слово 1>>\|<<запрещённое слово 2>>\|<<запрещённое слово 3>>" <файл>
```

- [ ] Не найдено `<<запрещённое слово 1>>`
- [ ] Не найдено `<<запрещённое слово 2>>`
- [ ] Не найдено `<<запрещённое слово 3>>`
- [ ] Не найдено `<<обещание гарантии / возврата / результата>>`, если такого правила нет в продукте

### ✅ Обязательные проверки содержания

- [ ] Регалия эксперта: указана **только в утверждённой формулировке** из `MEMORY.md`
- [ ] Архитектура продуктов: цены / тарифы / состав соответствуют `MEMORY.md` «Продуктовый факт-лист»
- [ ] Продукты, которые в этом запуске **не продаём** — не упоминаются как доступные
- [ ] Ключевые даты (эфир / старт потока / дедлайн) совпадают с `MEMORY.md`
- [ ] Гарантии / возврат / money-back — упоминаются **только если это публичный оффер**
- [ ] Каналы и активы, которые отложены, — не указаны как живые

### 📁 Проверка файла

- [ ] Файл реально сохранён по пути из брифа
- [ ] Размер файла > 0 байт
- [ ] Формат соответствует брифу (`.md` / `.html` / `.png` / …)

### 📝 Фиксация в памяти

- [ ] Записал в `MEMORY.md` строку: `Сдано: [имя_файла], [дата], статус: на апруве`

### 📤 Доставка

- [ ] Файл отправлен в чат владельца
- [ ] Подпись к файлу написана (одна строка)
