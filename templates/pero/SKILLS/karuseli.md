# Навык: Карусели

## Что я выдаю
Текст каждого слайда + однострочное ТЗ на визуал. Обычно 5–8 слайдов.

```
СЛАЙД 1 (обложка): «5 ошибок, из-за которых маникюр живёт неделю» · 📷 фон: крупно руки
СЛАЙД 2: ошибка 1 + почему · 📷 иллюстрация/фото
...
СЛАЙД N (финал): вывод + CTA «сохрани и запишись» · 📷 логотип/мастер
```

## Правила

- **Обложка = 80% успеха**: цифра, боль или провокация. По обложке
  должно быть ясно, зачем листать.
- Один слайд — одна мысль, максимум 3 строки текста на слайде.
- Нарастание: самое интересное не на 2-м слайде, а к середине —
  иначе дочитываемость падает.
- Предпоследний слайд — мягкая подводка к услуге, последний — CTA.
- «Сохраняемость» — главная метрика карусели: чек-листы, инструкции,
  «покажи мастеру» работают лучше всего.

## Рабочие форматы для локального бизнеса

- N ошибок / N мифов об услуге
- Чек-лист подготовки к процедуре/тренировке
- «Как выбрать [мастера/студию/процедуру] и не пожалеть»
- Прайс-гид: что входит в услугу по шагам
- История клиента: путь от проблемы к результату (с разрешения)
