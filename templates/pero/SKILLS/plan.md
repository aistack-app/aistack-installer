# Навык: Контент-план

## Что я выдаю
Таблица на неделю (по запросу — месяц): день, формат, тема, статус.

```
ПН · пост (экспертный) · «почему гель-лак скалывается» · черновик готов
ВТ · сторис-серия · окошки недели + опрос · раскадровка готова
СР · Reels · процесс-залипалка, услуга X · сценарий готов
ЧТ · — (день без публикации — это нормально)
ПТ · пост (продающий) · акция выходных из BUSINESS.md · черновик готов
СБ · сторис · живое из студии · идея
ВС · карусель · чек-лист ухода · черновик готов
```

## Принципы

- **Ритм важнее объёма**: лучше стабильные 4 единицы в неделю, чем
  10 на этой и ноль на следующей. Ритм согласую с владельцем один раз.
- Чередование типов: продающий максимум 1 из 3.
- План опирается на: акции из BUSINESS.md, события студии (от
  владельца/Часов), частые вопросы клиентов (от Голоса через Хозяина),
  удачные форматы из MEMORY.md.
- План — живой: владелец сказал «в среду привоз новых материалов» →
  среда переигрывается под инфоповод.

## Процесс

1. Собираю вводные (акции, события, прошлые удачи).
2. Выдаю план-сетку владельцу на согласование.
3. После «ок» — готовлю контент по плану заранее, не в день публикации.
4. В конце недели: что вышло/что нет → MEMORY.md.
