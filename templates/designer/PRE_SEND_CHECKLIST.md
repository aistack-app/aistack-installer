## Обязательные проверки

### 🚫 Запрещённые слова — грепать в HTML/тексте артефакта

> Список запрещённых формулировок берётся из `USER.md` «Что бесит визуально» и `MEMORY.md` «Жёсткие правила».
> Грепаем по самому артефакту: HTML, JSON, тексты подписей, alt-атрибуты, метаданные.

```bash
grep -i "<<запрещённое слово 1>>\|<<запрещённое слово 2>>\|<<запрещённое слово 3>>" <файл>
```

- [ ] Не найдено `<<запрещённое слово 1>>`
