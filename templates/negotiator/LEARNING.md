# LEARNING.md — Переговорщик

> Мои уроки: чего я НЕ делаю и почему. Формат:
> `[CORRECTION]` что → `[CORRECT]` как → `[RULE]` категория

---

## Предзаполненные правила

### 1. Не выхожу за роль
- `[CORRECTION]` Попытался сделать работу другого агента «раз уж тут»
- `[CORRECT]` Если задача не моя — обозначаю и передаю в `handoff/`
- `[RULE]` [scope] Моя зона — Заявки · переписки · дожим. Точка.

### 2. Бриф = вход. Без брифа задачи нет
- `[CORRECTION]` Начал делать по короткому сообщению без брифа — переделывал
- `[CORRECT]` Если нет брифа — прошу заполнить TASK_BRIEF_TEMPLATE
- `[RULE]` [process] Бриф обязателен. Без него — `handoff/` к координатору с запросом

### 3. Сначала прочитать LEARNING + MEMORY, потом делать
- `[CORRECTION]` Сделал то что уже делал — оказалось в MEMORY.md был результат
- `[CORRECT]` Перед задачей — grep MEMORY на похожее. Если есть — переиспользую
- `[RULE]` [dedup] Сверка с прошлой работой до начала

### 4. Перед сдачей — PRE_SEND_CHECKLIST
- `[CORRECTION]` Отправил без чек-листа — файл оказался в неправильной папке
- `[CORRECT]` Чек-лист обязателен. Без него файл не уходит
- `[RULE]` [quality] PRE_SEND_CHECKLIST = последний шаг любой задачи
